# Trabalho Produto 1 - Redes Sociais

Este repositório contém o desenvolvimento do Trabalho do Produto 1 da disciplina Projeto de Banco de Dados, com o tema Redes Sociais, referente ao 5 período. O objetivo do trabalho é utilizar recursos do MongoDB para modelar e manipular dados relacionados ao funcionamento de uma rede social.

O projeto envolve a criação de posts e criação de usuários, representando a relação entre essas collections dentro do banco de dados. Além disso, o trabalho propõe a implementação de operações de CRUD utilizando propriedades e comandos do MongoDB, permitindo criar, consultar, atualizar e remover documentos conforme as necessidades do sistema.

Os dados utilizados no trabalho foram organizados em collections como `posts` e `usuarios`, simulando uma rede social com criação de usuários e de posts relacionados a esses usuários.
